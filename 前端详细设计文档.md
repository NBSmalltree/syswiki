# "系统百科"平台前端详细设计文档

## 1. 文档说明

本文档基于《系统概要设计文档》，对前端工程进行详细设计，涵盖工程结构、路由规划、各功能模块组件设计、状态管理、API集成层及类型定义，可直接指导前端开发编码。

**技术栈：** Vue 3 + TypeScript + Vite + Element Plus + ECharts 5.x + md-editor-v3 + Pinia

## 3. 路由设计

### 3.1 路由表

```typescript
// src/router/index.ts
import { createRouter, createWebHistory } from 'vue-router'

const routes = [
  {

        path: '',

    component: () => import('@/views/admin/SpaceAdmin.vue'),

const router = createRouter({
  history: createWebHistory(),
  routes
})

// 路由守卫：进入空间页面前校验systemId有效性
router.beforeEach(async (to, from, next) => {
  if (to.meta.requiresSpace) {
    const systemId = to.params.systemId as string
    // 可在此校验systemId是否有效，无效则跳转门户
    // const valid = await spaceStore.validateSpace(systemId)
    // if (!valid) return next('/portal')
  }
  next()
})

export default router

  ├── /space/:systemId/ai-chat       → AI智能问答

            />

            </el-button>
          </div>

        </div>

        <!-- 渲染后的完整SQL -->
        <div v-if="renderedSql[sql.sqlId]" class="rendered-sql">
          <el-divider />

```

**核心功能点：**

1. **分类筛选：** 通过RadioGroup切换SQL分类，前端过滤或后端查询
2. **参数化SQL：**
   - SQL模板中包含占位符（如 `:billNo`）
   - 组件自动解析占位符，为每个参数生成输入框
   - 用户输入参数值后，点击"生成SQL"调用后端 `POST .../sql-lib/{sqlId}/render`
   - 后端返回替换后的完整SQL
3. **一键复制：** CopyButton组件封装 `navigator.clipboard.writeText()`，复制成功后显示提示

**CopyButton.vue 组件接口：**

**useSSE.ts 组合式函数设计：**

```typescript
// src/composables/useSSE.ts
export function useSSE() {
  const loading = ref(false)
  const currentContent = ref('')

  const sendMessage = async (
    systemId: string,
    message: string,
    model: 'flash' | 'think',
    onChunk: (chunk: string) => void,
    onDone: () => void,
    onError: (error: string) => void
  ) => {
    loading.value = true
    currentContent.value = ''

    try {
      const response = await fetch(`/api/spaces/${systemId}/ai/chat`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ message, model })
      })

      const reader = response.body?.getReader()
      const decoder = new TextDecoder()

      while (reader) {
        const { done, value } = await reader.read()
        if (done) break

        const text = decoder.decode(value)
        // 解析SSE格式: data: {...}\n\n
        const lines = text.split('\n')
        for (const line of lines) {
          if (line.startsWith('data: ')) {
            const data = JSON.parse(line.slice(6))
            if (data.content) {
              currentContent.value += data.content
              onChunk(data.content)
            }
            if (data.done) {
              onDone()
            }
          }
        }
      }
    } catch (err) {
      onError('请求失败，请稍后重试')
    } finally {
      loading.value = false
    }
  }

  return { loading, currentContent, sendMessage }
}
```

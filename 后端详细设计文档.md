## 1. 文档说明

本文档基于《系统概要设计文档》，对后端工程进行详细设计，涵盖工程结构、完整DDL建表脚本、RESTful API接口定义（含请求/响应示例）、核心Service层设计、RAG工作流方案及LangChain4j集成设计，可直接指导后端开发编码。

## 3. 完整DDL建表脚本

### 3.1 系统空间主表

```sql
-- 系统空间主表
CREATE TABLE SYS_ENCY_SPACE (
    SYSTEM_ID       VARCHAR2(32)    NOT NULL,
    SYSTEM_NAME     VARCHAR2(128)   NOT NULL,
    SYSTEM_CODE     VARCHAR2(32)    NOT NULL,
    OWNER           VARCHAR2(64)    NOT NULL,
    DESCRIPTION     VARCHAR2(512),
    STATUS          VARCHAR2(16)    DEFAULT 'ACTIVE' NOT NULL,
    CREATE_TIME     DATE            DEFAULT SYSDATE NOT NULL,
    UPDATE_TIME     DATE            DEFAULT SYSDATE,
    CONSTRAINT PK_SYS_ENCY_SPACE PRIMARY KEY (SYSTEM_ID),
    CONSTRAINT UK_SPACE_CODE UNIQUE (SYSTEM_CODE)
);

COMMENT ON TABLE SYS_ENCY_SPACE IS '系统百科-空间主表';
COMMENT ON COLUMN SYS_ENCY_SPACE.SYSTEM_ID IS '系统唯一编号（主键）';
COMMENT ON COLUMN SYS_ENCY_SPACE.SYSTEM_NAME IS '系统名称（如：超级网上银行）';
COMMENT ON COLUMN SYS_ENCY_SPACE.SYSTEM_CODE IS '系统代号（如：SIB），唯一约束';
COMMENT ON COLUMN SYS_ENCY_SPACE.OWNER IS '系统负责人';
COMMENT ON COLUMN SYS_ENCY_SPACE.DESCRIPTION IS '系统简要描述';
COMMENT ON COLUMN SYS_ENCY_SPACE.STATUS IS '状态：ACTIVE-正常，DISABLED-停用';
COMMENT ON COLUMN SYS_ENCY_SPACE.CREATE_TIME IS '创建时间';
COMMENT ON COLUMN SYS_ENCY_SPACE.UPDATE_TIME IS '最近更新时间';

CREATE INDEX IDX_SPACE_OWNER ON SYS_ENCY_SPACE(OWNER);
```

## 4. 统一响应封装

### 4.1 Result.java

```java
package com.syswiki.model.vo;

import lombok.Data;

@Data
public class Result<T> {
    private int code;
    private String message;
    private T data;

    public static <T> Result<T> success(T data) {
        Result<T> r = new Result<>();
        r.setCode(200);
        r.setMessage("success");
        r.setData(data);
        return r;
    }

    public static <T> Result<T> error(int code, String message) {
        Result<T> r = new Result<>();
        r.setCode(code);
        r.setMessage(message);
        return r;
    }

    public static Result<Void> error(ErrorCode errorCode) {
        Result<Void> r = new Result<>();
        r.setCode(errorCode.getCode());
        r.setMessage(errorCode.getMessage());
        return r;
    }
}
```

### 4.3 GlobalExceptionHandler.java

```java
package com.syswiki.exception;

import com.syswiki.model.vo.Result;
import lombok.extern.slf4j.Slf4j;
import org.springframework.web.bind.MethodArgumentNotValidException;
import org.springframework.web.bind.annotation.ExceptionHandler;
import org.springframework.web.bind.annotation.RestControllerAdvice;

@Slf4j
@RestControllerAdvice
public class GlobalExceptionHandler {

    @ExceptionHandler(BizException.class)
    public Result<Void> handleBizException(BizException e) {
        log.warn("业务异常: code={}, msg={}", e.getCode(), e.getMessage());
        return Result.error(e.getCode(), e.getMessage());
    }

    @ExceptionHandler(MethodArgumentNotValidException.class)
    public Result<Void> handleValidation(MethodArgumentNotValidException e) {
        String msg = e.getBindingResult().getAllErrors().get(0).getDefaultMessage();
        return Result.error(ErrorCode.PARAM_INVALID.getCode(), msg);
    }

    @ExceptionHandler(Exception.class)
    public Result<Void> handleException(Exception e) {
        log.error("系统异常", e);
        return Result.error(ErrorCode.INTERNAL_ERROR);
    }
}
```

#### 5.2.3 保存/更新模块内容

```
PUT /api/spaces/{systemId}/contents/{moduleType}
```

**请求体：**

```json
{
  "mdContent": "# 系统简介\n\n超级网上银行系统是我行核心渠道系统...\n\n## 技术栈\n\n- Java 11\n- Spring Boot 2.7\n- OceanBase",
  "operator": "张三"
}
```

**后端处理流程：**
1. 校验systemId和moduleType有效性
2. 敏感词扫描（如有则拒绝保存）
3. 查询是否存在该模块内容
   - 存在：更新MD_CONTENT，VERSION自增，更新OPERATOR和UPDATE_TIME
   - 不存在：插入新记录，VERSION=1
4. 保存版本历史到SYS_ENCY_CONTENT_VERSION
5. 异步触发向量化（文本切片→Embedding→存入向量库）
6. 返回更新后的内容对象

**响应：** 返回更新后的ContentVO。

#### 5.2.4 Markdown文件导入

```
POST /api/spaces/{systemId}/contents/import
Content-Type: multipart/form-data
```

**请求参数：**

| 参数 | 类型 | 说明 |
|------|------|------|
| file | File | .md文件 |
| operator | string | 操作人 |

**后端处理流程：**
1. 读取上传的.md文件内容
2. 调用MarkdownParserService解析标题层级
3. 按二级/三级标题拆分为多个MODULE_TYPE段落
4. 逐个模块更新SYS_ENCY_CONTENT表
5. 每个模块保存版本历史
6. 异步触发全量向量化

**Markdown解析规则：**

```java
/**
 * 解析逻辑：
 * - 一级标题(#): 文档标题，忽略
 * - 二级标题(##): 对应MODULE_TYPE
 *   - "系统简介" / "技术栈" → INTRO
 *   - "测试环境架构" → ARCH_TEST
 *   - "生产环境架构" → ARCH_PROD
 *   - "服务器" → SERVER
 *   - "网络" → NETWORK
 *   - "数据库" → DATABASE
 *   - "接入指南" / "快速接入" → GUIDE
 * - 三级标题(###): 作为模块内的子章节保留
 * - 未匹配的标题: 归入上一个已识别的MODULE_TYPE
 */
```

```java
/**
 * 参数化SQL渲染：
 * 1. 读取SQL_TEMPLATE
 * 2. 遍历params Map
 * 3. 将 :paramName 替换为对应的值
 * 4. 对字符串类型的值加上单引号（防SQL注入仅限展示用途，不做实际执行）
 * 5. 注意：此SQL仅供人工复制使用，平台不执行任何SQL
 */
public String renderSql(String template, Map<String, String> params) {
    String result = template;
    for (Map.Entry<String, String> entry : params.entrySet()) {
        String placeholder = ":" + entry.getKey();
        String value = "'" + entry.getValue().replace("'", "''") + "'";
        result = result.replace(placeholder, value);
    }
    return result;
}
```

### 6.3 MarkdownParserService.java

```java
@Service
public class MarkdownParserService {

    /**
     * 解析Markdown文件，按标题层级拆分为多个模块内容
     *
     * @param markdownContent Markdown全文
     * @return Map<ModuleType, 内容片段>
     */
    public Map<String, String> parseByHeadings(String markdownContent) {
        Map<String, String> result = new LinkedHashMap<>();
        String[] lines = markdownContent.split("\n");

        String currentModule = null;
        StringBuilder currentContent = new StringBuilder();

        for (String line : lines) {
            if (line.startsWith("## ")) {
                // 二级标题 → 识别模块类型
                if (currentModule != null) {
                    result.put(currentModule, currentContent.toString().trim());
                }
                String heading = line.substring(3).trim();
                currentModule = matchModuleType(heading);
                currentContent = new StringBuilder();
            } else if (currentModule != null) {
                currentContent.append(line).append("\n");
            }
        }

        // 最后一个模块
        if (currentModule != null) {
            result.put(currentModule, currentContent.toString().trim());
        }

        return result;
    }

    /**
     * 标题文本匹配MODULE_TYPE
     */
    private String matchModuleType(String heading) {
        String h = heading.toLowerCase();
        if (h.contains("简介") || h.contains("技术栈")) return "INTRO";
        if (h.contains("测试") && h.contains("架构")) return "ARCH_TEST";
        if (h.contains("生产") && h.contains("架构")) return "ARCH_PROD";
        if (h.contains("服务器") || h.contains("基础设施")) return "SERVER";
        if (h.contains("网络") || h.contains("防火墙") || h.contains("负载")) return "NETWORK";
        if (h.contains("数据库") || h.contains("DB")) return "DATABASE";
        if (h.contains("接入") || h.contains("指南")) return "GUIDE";
        return null; // 未匹配则忽略
    }

    /**
     * 合并多个模块内容为完整Markdown
     */
    public String mergeModules(Map<String, String> modules) {
        StringBuilder sb = new StringBuilder();
        String[] order = {"INTRO", "ARCH_TEST", "ARCH_PROD", "SERVER",
                          "NETWORK", "DATABASE", "GUIDE"};

        for (String type : order) {
            String content = modules.get(type);
            if (content != null && !content.isEmpty()) {
                sb.append("## ").append(getModuleHeading(type)).append("\n\n");
                sb.append(content).append("\n\n");
            }
        }
        return sb.toString().trim();
    }

    private String getModuleHeading(String type) {
        switch (type) {
            case "INTRO": return "系统简介与技术栈";
            case "ARCH_TEST": return "测试环境架构";
            case "ARCH_PROD": return "生产环境架构";
            case "SERVER": return "服务器配置";
            case "NETWORK": return "网络策略管理";
            case "DATABASE": return "数据库配置";
            case "GUIDE": return "快速接入指南";
            default: return type;
        }
    }
}
```

### 7.2 向量存储设计（SysWikiEmbeddingStoreFactory.java）

```java
package com.syswiki.rag;

import dev.langchain4j.data.embedding.Embedding;
import dev.langchain4j.data.segment.TextSegment;
import dev.langchain4j.store.embedding.EmbeddingMatch;
import dev.langchain4j.store.embedding.EmbeddingSearchRequest;
import dev.langchain4j.store.embedding.EmbeddingSearchResult;
import dev.langchain4j.store.embedding.EmbeddingStore;
import dev.langchain4j.store.embedding.in.InProcessEmbeddingStore;
import org.springframework.stereotype.Component;

import java.util.List;
import java.util.concurrent.ConcurrentHashMap;

/**
 * 向量存储工厂
 * 每个systemId维护独立的向量存储空间，实现多系统隔离
 *
 * 首期方案：使用LangChain4j内置的InProcessEmbeddingStore（内存+持久化）
 * 后续可替换为PgVector或Qdrant，仅需修改此类
 */
@Component
public class SysWikiEmbeddingStoreFactory {

    /**
     * 按systemId隔离的向量存储映射
     */
    private final ConcurrentHashMap<String, EmbeddingStore<TextSegment>> stores
        = new ConcurrentHashMap<>();

    /**
     * 获取或创建指定系统的向量存储
     */
    public EmbeddingStore<TextSegment> getStore(String systemId) {
        return stores.computeIfAbsent(systemId, id -> {
            // 首期：内存存储
            // 后续可替换为：
            //   PgVectorEmbeddingStore.builder().build()
            //   QdrantEmbeddingStore.builder().build()
            return InProcessEmbeddingStore.builder().build();
        });
    }

    /**
     * 添加向量到指定系统空间
     */
    public void add(String systemId, Embedding embedding, TextSegment segment) {
        getStore(systemId).add(embedding, segment);
    }

    /**
     * 从指定系统空间检索相似内容
     */
    public List<EmbeddingMatch<TextSegment>> search(
            String systemId, Embedding queryEmbedding, int maxResults) {
        EmbeddingSearchRequest request = EmbeddingSearchRequest.builder()
            .queryEmbedding(queryEmbedding)
            .maxResults(maxResults)
            .minScore(0.6)  // 相似度阈值
            .build();

        EmbeddingSearchResult<TextSegment> result =
            getStore(systemId).search(request);
        return result.matches();
    }

    /**
     * 清除指定系统的全部向量（用于全量重建）
     */
    public void clearSystem(String systemId) {
        stores.remove(systemId);
    }
}
```

### 7.5 AI问答Service（AiChatService.java）

```java
package com.syswiki.service;

import com.syswiki.model.entity.SysEncySpace;
import com.syswiki.rag.PromptTemplates;
import com.syswiki.rag.SysWikiContentRetriever;
import com.syswiki.rag.SysWikiEmbeddingStoreFactory;
import dev.langchain4j.data.segment.TextSegment;
import dev.langchain4j.model.chat.StreamingChatLanguageModel;
import dev.langchain4j.model.chat.response.ChatResponse;
import dev.langchain4j.model.chat.response.StreamingChatResponseHandler;
import dev.langchain4j.rag.content.Content;
import dev.langchain4j.rag.query.Query;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.beans.factory.annotation.Qualifier;
import org.springframework.stereotype.Service;

import java.util.List;
import java.util.stream.Collectors;

@Slf4j
@Service
@RequiredArgsConstructor
public class AiChatService {

    @Qualifier("flashModel")
    private final StreamingChatLanguageModel flashModel;

    @Qualifier("thinkModel")
    private final StreamingChatLanguageModel thinkModel;

    private final SysWikiContentRetriever contentRetriever;
    private final SpaceService spaceService;

    /**
     * 执行RAG问答
     *
     * @param systemId   系统空间ID
     * @param message    用户提问
     * @param modelType  模型档位：flash/think
     * @param callback   流式回调
     */
    public void chat(String systemId, String message, String modelType,
                     StreamCallback callback) {
        // 1. 获取系统信息
        SysEncySpace space = spaceService.getById(systemId);
        if (space == null) {
            callback.onError(new RuntimeException("系统空间不存在"));
            return;
        }

        // 2. RAG检索：在当前systemId的向量空间中检索相关内容
        List<Content> relevantContents =
            contentRetriever.retrieve(new Query(message), systemId);

        // 3. 构建知识库上下文
        String knowledgeContext = relevantContents.stream()
            .map(Content::text)
            .collect(Collectors.joining("\n\n---\n\n"));

        // 4. 组装Prompt
        String systemPrompt = String.format(
            PromptTemplates.SYSTEM_PROMPT,
            space.getSystemName(),
            space.getOwner(),
            knowledgeContext
        );

        if ("think".equals(modelType)) {
            systemPrompt += "\n\n" + PromptTemplates.THINK_ADDITION;
        }

        // 5. 选择模型
        StreamingChatLanguageModel model = "think".equals(modelType)
            ? thinkModel : flashModel;

        // 6. 流式调用
        String fullPrompt = systemPrompt + "\n\n用户提问：" + message;

        model.chat(fullPrompt, new StreamingChatResponseHandler() {
            @Override
            public void onPartialResponse(String token) {
                callback.onToken(token);
            }

            @Override
            public void onCompleteResponse(ChatResponse response) {
                log.info("AI问答完成: systemId={}, model={}, tokens={}",
                    systemId, modelType, response.tokenUsage());
                callback.onComplete();
            }

            @Override
            public void onError(Throwable error) {
                log.error("AI问答异常: systemId={}", systemId, error);
                callback.onError(error);
            }
        });
    }

    /**
     * 流式回调接口
     */
    public interface StreamCallback {
        void onToken(String token);
        void onComplete();
        void onError(Throwable t);
    }
}
```

### 7.6 向量同步Service（VectorSyncService.java）

```java
package com.syswiki.service;

import com.syswiki.rag.SysWikiEmbeddingStoreFactory;
import dev.langchain4j.data.document.Document;
import dev.langchain4j.data.document.splitter.DocumentSplitter;
import dev.langchain4j.data.segment.TextSegment;
import dev.langchain4j.model.embedding.EmbeddingModel;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.scheduling.annotation.Async;
import org.springframework.stereotype.Service;

@Slf4j
@Service
@RequiredArgsConstructor
public class VectorSyncService {

    private final SysWikiEmbeddingStoreFactory storeFactory;
    private final EmbeddingModel embeddingModel;
    private final DocumentSplitter textSplitter;

    /**
     * 异步同步内容到向量库
     * 当Markdown内容更新时调用
     */
    @Async
    public void syncContent(String systemId, String moduleType, String mdContent) {
        try {
            log.info("开始向量化: systemId={}, module={}", systemId, moduleType);

            // 1. 构建Document对象
            Document document = Document.from(mdContent);

            // 2. 文本切片
            List<TextSegment> segments = textSplitter.split(document);

            // 3. 为每个片段添加元数据
            segments.forEach(seg -> {
                seg.metadata().put("systemId", systemId);
                seg.metadata().put("moduleType", moduleType);
            });

            // 4. 批量Embedding并存储
            for (TextSegment segment : segments) {
                var embedding = embeddingModel.embed(segment).content();
                storeFactory.add(systemId, embedding, segment);
            }

            log.info("向量化完成: systemId={}, module={}, chunks={}",
                systemId, moduleType, segments.size());
        } catch (Exception e) {
            log.error("向量化失败: systemId={}, module={}", systemId, moduleType, e);
        }
    }

    /**
     * 全量重建指定系统的向量空间
     * 用于系统初始化或数据修复
     */
    @Async
    public void rebuildAll(String systemId, List<String> allContents) {
        log.info("开始全量重建向量: systemId={}", systemId);

        // 1. 清除旧向量
        storeFactory.clearSystem(systemId);

        // 2. 重新向量化所有内容
        for (String content : allContents) {
            Document document = Document.from(content);
            List<TextSegment> segments = textSplitter.split(document);

            for (TextSegment segment : segments) {
                segment.metadata().put("systemId", systemId);
                var embedding = embeddingModel.embed(segment).content();
                storeFactory.add(systemId, embedding, segment);
            }
        }

        log.info("全量重建完成: systemId={}", systemId);
    }
}
```

## 11. ID生成器

```java
package com.syswiki.util;

import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;
import java.util.concurrent.atomic.AtomicLong;

/**
 * 简易ID生成器
 * 格式：前缀 + 日期 + 序号（如 SP20260101000001）
 */
public class IdGenerator {

    private static final AtomicLong SEQUENCE = new AtomicLong(0);
    private static final DateTimeFormatter DF =
        DateTimeFormatter.ofPattern("yyyyMMdd");

    public static String nextId() {
        return nextId("ID");
    }

    public static String nextId(String prefix) {
        String date = LocalDateTime.now().format(DF);
        long seq = SEQUENCE.incrementAndGet() % 1000000;
        return prefix + date + String.format("%06d", seq);
    }

    // 使用示例：
    // IdGenerator.nextId("SP")  → SP20260528000001
    // IdGenerator.nextId("CT")  → CT20260528000001
    // IdGenerator.nextId("TL")  → TL20260528000001
    // IdGenerator.nextId("SQL") → SQL2026052800001
}
```
